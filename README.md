# The Funded Trader Course

A free, interactive course on prop-firm trading — built for a complete beginner starting from zero.

**11 modules** covering prop-firm mechanics, market basics, risk management, chart reading, strategy, trading psychology, challenge tactics, firm selection, and a full worked example of passing a challenge trade by trade.

**Built-in tools:**
- Position size calculator
- Drawdown recovery calculator
- Challenge simulator (1,000 randomised attempts with equity curve visualisation)

**Interactive quizzes** with explanations at the end of every module.

## How to use

Open `index.html` in any browser. No installation, no dependencies, no internet required (except for Google Fonts).
